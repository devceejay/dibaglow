# DibaGlow - Medical Spa Website

DibaGlow is a self-initiated concept project for a luxury wellness and medical spa brand. The website was designed and built using HTML, CSS, and JavaScript, combining calming visuals with intuitive navigation to deliver a serene, user-friendly digital experience.

## Table of Contents:

## Overview:

DibaGlow showcases a polished, user-centered website that highlights services, builds trust with testimonials, answers client questions through a clear FAQ section, and encourages conversions with an accessible contact form. It was created to demonstrate my ability to bring brand identities to life through thoughtful design and clean development practices.

## 📌 Features:

**Modern, Elegant Design:** The site uses soft visuals, neutral tones, and calming animations to align with the brand's wellness and luxury identity.

**Service Cards:** Each spa service is displayed in a well-structured and visually appealing card format, with detailed information.

**FAQ Section:** An accordion-style FAQ section to answer common client questions.

**Contact Form:** A user-friendly contact form for inquiries and appointments.

**Fully Responsive Design:** Ensures a seamless experience across all screen sizes.

**Testimonials Section:** Showcasing positive client feedback.

Pricing section with clear service breakdown.

Newsletter subscription section for client engagement.

## 🔧 Tech Stack:

**HTML5:** For structuring the website content.

**CSS3:** Custom stylesheets for layout, colors, and visual elements.

**JavaScript:** For adding interactivity such as FAQ interaction.
